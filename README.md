# node-firebase-test
Is a simple exercise to create, get and delete contacts using Firebase Databse with a simple using UI 

##Stack 
-Frrebase
-Node js
-Express
-Handlebars
-Bootstrap

##Envinroment setup

To install dependencies, run
``` bash
npm install
```
## Start local server

``` bash
npm run dev
```
